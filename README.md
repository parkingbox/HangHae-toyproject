### ✅ Key Features
1. 현재 즐길 수 있는 축제 정보 케러셀 슬라이드 <br>
2. 날짜별 축제 정렬 후 보여주기 기능 <br>
3. 포스터 클릭 시 해당 포스터의 상세 페이지로 이동 <br>
4. 상세페에지 접속 시 데이터(포스터, 이름, 위치, 날짜, 전화번호, 지도) 보여줌 <br>
5. 상세페이지별 댓글 달기, 삭제 기능 <br>
